webrender::device::gl

[−][src]Static webrender::device::gl::GPU_BYTES_ALLOCATED

static GPU_BYTES_ALLOCATED: AtomicUsize

Tracks the total number of GPU bytes allocated across all WebRender instances.

Assuming all WebRender instances run on the same thread, this doesn't need to be atomic per se, but we make it atomic to satisfy the thread safety invariants in the type system. We could also put the value in TLS, but that would be more expensive to access.
